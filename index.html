About Me

Hello! My name is Dharma Teja, and I am a passionate web designer and front-end developer. With a keen eye for design and a strong foundation in web technologies, I strive to create visually appealing and highly functional websites. My journey in web development began with a fascination for the internet and its limitless potential for creativity and innovation. Over the years, I have honed my skills in HTML, CSS, JavaScript, and various front-end frameworks, allowing me to turn complex ideas into user-friendly digital experiences. I am dedicated to staying up-to-date with the latest industry trends and best practices, ensuring that the websites I create are not only aesthetically pleasing but also optimized for performance and accessibility. My goal is to deliver engaging and seamless user experiences that meet the needs of both clients and end-users. When I'm not coding or designing, you can find me exploring new design concepts, learning about emerging technologies, or collaborating with other professionals in the field to continuously improve my craft. Thank you for taking the time to learn a bit about me. I look forward to the opportunity to bring your web projects to life!

EXPENSE TRACKER

The Expense Tracker is a dynamic web application designed to help users efficiently manage their personal finances. Built using HTML, CSS, and JavaScript, this project offers a user-friendly interface where users can easily add, view, and delete their expenses. The application features real-time updates and visual representations of spending patterns, enabling users to gain better insights into their financial habits. This project demonstrates my proficiency in front-end development and my ability to create practical, interactive web applications.

PG life

PG Life is a comprehensive web application designed to simplify the search for paying guest accommodations. Created using HTML, CSS, Bootstrap, and a touch of JavaScript, this project includes four key pages: Home, Property List, Property Detail, and Dashboard. The Home page offers a welcoming introduction to the service, while the Property List page displays available accommodations in an organized manner. The Property Detail page provides in-depth information about each listing, and the Dashboard page allows users to manage their profiles and preferences. This project highlights my ability to build responsive and user-friendly web applications with modern front-end technologies.
